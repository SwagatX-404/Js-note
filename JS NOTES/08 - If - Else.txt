CONTROL STRUCTURES IN JAVASCRIPT
‚Üí Programming is not always top-to-bottom (sequential)
‚Üí Real-world logic needs decisions & repetitions
‚Üí For that we need CONTROL STRUCTURES

Two Types:
1. Conditional Statements ‚Üí Take decisions
2. Loops ‚Üí Repeat code

Today‚Äôs Topic: CONDITIONAL STATEMENTS
(Part 1 ‚Äì if, if-else, if-else-if-else)

1. if Statement
Syntax:
if (condition) {
    // code executes ONLY if condition is true
}

Example 1: Night Club Entry
let age = 20;

if (age >= 18) {
    console.log("Welcome to the club üéâ");
}
// Output: Welcome to the club

If age = 15 ‚Üí No output (condition false)

Example 2: Free Shipping on Amazon
let cartTotal = 12000;

if (cartTotal >= 10000) {
    console.log("You qualified for FREE shipping! üöö");
}
// Output: You qualified for FREE shipping!

2. if-else Statement
Syntax:
if (condition) {
    // runs if true
} else {
    // runs if false
}

Example 1: Airport Lounge
let isVIP = true;

if (isVIP) {
    console.log("Welcome to VIP Lounge ‚ú®");
} else {
    console.log("Welcome to Regular Lounge");
}
// Output: Welcome to VIP Lounge

If isVIP = false ‚Üí Regular Lounge

Example 2: Bank Withdrawal
let balance = 1000;
let withdrawAmount = 700;

if (balance >= withdrawAmount) {
    console.log("Transaction Successful ‚úÖ");
} else {
    console.log("Insufficient Balance ‚ùå");
}
// Output: Transaction Successful

If withdrawAmount = 1500 ‚Üí Insufficient Balance

3. if else if else (Multiple Conditions)
Used when you have more than 2 possibilities

Syntax:
if (condition1) {
    // block 1
} else if (condition2) {
    // block 2
} else if (condition3) {
    // block 3
} else {
    // default block
}

Real Example: Indian Railways Ticket Price
let age = 14;

if (age < 12) {
    console.log("Child Ticket Price üë∂");
} else if (age >= 65) {
    console.log("Senior Citizen Ticket Price üßì");
} else {
    console.log("Adult Ticket Price üßë");
}
// Output: Adult Ticket Price

Test yourself:
age = 8   ‚Üí Child Ticket Price
age = 70  ‚Üí Senior Citizen Ticket Price
age = 25  ‚Üí Adult Ticket Price

Important Flow Rules:
‚Üí Checks conditions from TOP to BOTTOM
‚Üí Only FIRST true block executes
‚Üí If none true ‚Üí else runs (optional)
‚Üí else is always optional
‚Üí You can have many else if, but only one else

Quick Interview Table

Best Practice Tips:
‚Üí Always use curly braces { } even for one line
‚Üí Use strict equality === whenever possible
‚Üí Keep conditions simple and readable
‚Üí Indent properly ‚Äì helps in debugging
